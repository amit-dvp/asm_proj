Run this inside your project folder:

git commit -m "the first maman1 version"

See what will be committed
Code
git status

See what changed
Code
git diff

View commit history
Code
git log --oneline --graph

Revert to last commit : 
git reset --hard HEAD

git remote add origin https://github.com/amit-dvp/asm_proj

To verify it was added correctly : 

git remote -v

Add updated files : 
git push -u origin master
